Windows可以禁用的无用服务

自上世纪95年Windows95发布以来，Windows便毫无疑问地成为了PC端的霸主，统一着整个PC帝国。用户多，个人用户要娱乐日用新功能，企业用户又要性能安全稳定性，众口难调，最后只得一股脑地把有用没用的功能塞进去，让用户自己决定。

那么对于个人用户来说，毫无疑问有些功能是用不到的了。

Application Layer Gateway Service 参考 Win10 Application Layer Gateway Service服务有什么用

Connected User Experiences and Telemetry 参考 Win10 Connected User Experiences and Telemetry服务可以禁用吗

Diagnostic Policy Service Diagnostic Policy Service服务是什么（若无替代品不要禁用）

Diagnostic Service Host 基本和Diagnostic Policy Service/Diagnostic Service Host是同类,可以一起禁用

Diagnostic System Host 同上

Distributed Link Tracking Client 参考 Distributed Link Tracking Client是什么服务

Downloaded Maps Manager （供应用程序访问已下载地图的 Windows 服务。一般用户不会在本地下载地图吧）

Function Discovery Provider Host （当电脑加入域后，共享文件需要的一个服务，非域则禁用。企业用户常用，个人用户用不到）

Internet Connection Sharing (ICS) （无线共享相关服务，如不需要无线共享，热点等可关闭）

IPsec Policy Agent （防火墙相关，用来协商和配置IPSEC安全策略，一般用户用不上）

Link-Layer Topology Discovery Mapper （网络映射相关，一般用户用不上）

Microsoft App-V Client （应用程序虚拟化技术客户端，不使用虚拟机可禁用）

Microsoft iSCSI Initiator Service （iscsi远程连接服务，禁用）

Net.Tcp Port Sharing Service （TCP端口共享服务，个人用户一般用不上）

Offline Files 参考 Offline Files是什么服务 (脱机服务，个人用户一般用不上）

Performance Logs & Alerts （搜集计算机性能、日志、报警信息的服务项，没啥用，一般人也不会看）

Portable Device Enumerator Service 参考Portable Device Enumerator Service服务有什么用

Program Compatibility Assistant Service （Windows兼容性助理，Windows中微软30年的积淀带来的强大的兼容性使这个功能几乎用不到）

Quality Windows Audio Video Experience 参考 Quality Windows Audio Video Experience是什么服务

Remote Desktop Configuration （远程桌面服务，关）

Remote Desktop Services （远程桌面服务，关）

Remote Desktop Services UserMode Port Redirector （远程桌面服务，关）

Remote Registry 参考 Remote Registry是什么服务（远程注册表服务，默认禁用。）

Routing and Remote Access (软路由服务，禁用)

Secondary Logon （二次登录服务，建议禁用）

Server （局域网共享相关服务，不使用局域网共享文件的可以禁用 ）

Shared PC Account Manager （共享PC帐户管理器，可以禁用）

Shell Hardware Detection （提供自动播放事件通知，禁用）

Smart Card （智能卡服务，禁用）

Smart Card Device Enumeration Service （同上，还是服务于智能卡，禁用）

Smart Card Removal Policy （智能卡相关服务）

SSDP Discovery （发现网络上的UPnP设备，网络发现相关服务，不使用局域网可禁用）

SysMain 参考 SysMain是什么服务 (预读取服务 如配备高速硬盘禁用无影响）

SynTPEnh Caller Service （同步电话呼叫服务 禁用）

TCP/IP NetBIOS Helper （局域网共享相关服务，不用局域网共享文件可禁用）

Telephony （电话应用程序编程接口 (TAPI) 提供支持 ，拨号用户不要关（现在真的还有拨号用户了吗），否则禁用）

User Experience Virtualization Service （用户体验虚拟化服务，不需要禁用）

Windows Biometric Service （生物识别服务，不使用指纹、不使用人脸识别登录系统可以禁用）

Windows Search （Windows搜索服务，加快搜索速度，建议禁用，使用替代软件（如listary））

Windows 推送通知系统服务 （建议关闭，会狂占内存、CPU，但禁用有概率出现问题）

WinHTTP Web Proxy Auto-Discovery Service （默认手动，可禁用）

Xbox Live 身份验证管理器 （不玩xbox游戏禁用）

Xbox Live 游戏保存 （同上）

XboxNetApiSvc （同上）

零售演示服务 （用于演示模式，禁用）

自动时区更新程序 （网络时间同步，不需要可禁用）
